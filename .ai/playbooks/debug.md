# Debug Playbook

## Goal

Быстро локализовать и исправить дефект, не внося побочные регрессии.

## Steps

1. Локализовать проблему по слою и доменному флоу.
2. Подтвердить root cause воспроизводимым сценарием.
3. Проверить, не задет ли lock-контракт.
4. Сделать минимальный рабочий fix.
5. Прогнать quality-gates.
6. Проверить негативные и смежные сценарии.

## Checklist

- Root cause доказан, а не предположен.
- Fix минимальный и понятный.
- Смежные флоу не сломаны.
- PR содержит воспроизведение и проверку исправления.
